# Dataset of Creep Properties for Nuclear Structural Materials

Authors: Isshu Lee, John W. Merickel, Boopathy Kombaiah, Yalei Tang, Fei Xu, Rongjie Song, Aleksandar Vakanski

The dataset contains records of creep tests of nuclear structural materials. As advanced nuclear reactor designs move towards higher operating temperatures, creep research has become increasingly important. The focus is on understanding the influence of specimen size on creep properties, including the relationship between sub-sized and standard-sized specimen behavior under irradiation constraints. The dataset was curated from 50 peer-reviewed sources through an extensive literature review. The extracted data points are organized into a tabular format database containing 1,413 test records with 71 parameters, including material type and composition, manufacturing information, irradiation conditions, specimen dimensions, and creep properties. The dataset focuses on materials relevant to nuclear reactor applications, including austenitic stainless steels, reactor pressure vessel steels, Inconel alloys, and Zircaloy. Creep test types represented in the dataset include uniaxial creep tests, impression creep tests, small punch creep tests, small ring tests, and two-bar tests. The dataset is intended to enable researchers to perform studies on the influences of specimen size on creep properties, and to support machine learning and statistical modeling for creep rupture life prediction.
